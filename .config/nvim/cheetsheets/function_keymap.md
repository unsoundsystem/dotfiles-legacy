Function key mappings
=====================

Normal mode
-----------

- <F1> : help
- <F2> : nothing
- <F3> : :FloatermNew 
- <F4> : Insert date and time
- <F5> : undotree
- <F6> : lazygit(floaterm)
- <F7> : nothing
- <F8> : TagbarToggle
- <F9> : ranger(floaterm)
- <F10> : FloatermToggle
- <F11> ~ <F12> : reserved
